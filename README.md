# find

L'applicazione find consente di individuare il numero di occorrenze di un insieme di stringhe all'interno di un gruppo di file. Per determinare il numero di occorrenze (e la loro posizione) viene utilizzato l'Algoritmo di Knuth-Morris e Pratt.
